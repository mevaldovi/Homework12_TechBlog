# Installation

- _Clone this repo_ 

- `npm install`

- _open terminal_

- `mysql -u root -p`

- _enter mysql password if you have one_

- `source db/schema.sql`

- `quit`

- `npm run seed`

- `cd server.js`

- `node server.js`

- `cd ..`

- `npm start`

- _visit https://localhost:3001 to view in browser_
